# Do-re-sal-ma-ri

Een leuk spel waarbij je een fles laat draaien om een winnaar te kiezen.

## Hoe te spelen

1. Voer de namen in van de deelnemers, gescheiden door komma's
2. Klik op "Spin the bottle"
3. De fles draait en kiest een winnaar
